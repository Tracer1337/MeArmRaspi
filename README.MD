# MeArm V0.4 Raspberry Pi

### Wiring:

* PCA9685:
    * Servos
        * 0   -> Base
        * 1   -> Right
        * 2   -> Left
        * 3   -> Grapper

    * Raspberry-Pi
        * V+  -> No Connection
        * VCC -> 5V
        * SDA -> PIN 3 (GPIO 2 - SDA)
        * SCL -> PIN 5 (GPIO 3 - SCL)
        * OE  -> No Connection
        * GND -> GND

    * Power Supply
        * V+  -> 4V - 6V
        * GND -> GND
