# MeArm V0.4 Raspberry Pi

### Wiring:

PCA9685:
    * Servos
        1. Base
        1. Right
        1. Left
        1. Grapper

    * Raspberry-Pi
        1. V+  -> No Connection
        1. VCC -> 5V
        1. SDA -> PIN 3 (GPIO 2 - SDA)
        1. SCL -> PIN 5 (GPIO 3 - SCL)
        1. OE  -> No Connection
        1. GND -> GND

    * Power Supply
        1. V+  -> 4V - 6V
        1. GND -> GND
